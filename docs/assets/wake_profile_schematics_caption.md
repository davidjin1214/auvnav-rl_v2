Figure X. Schematic of the wake-generation profiles used in `scripts/generate_wake.py`: (a) single-cylinder configuration, (b) tandem-cylinder configuration with streamwise spacing `G/D = 3.5`, and (c) side-by-side cylinder configuration with transverse spacing `G/D = 3.5`. The outer rectangle in each panel denotes the full computational domain, the hatched blue region denotes the exported region of interest (ROI), and the filled circles denote cylinder positions. Arrows indicate the free-stream direction `U_\infty`. Domain and ROI dimensions are reported in metres in the upper-right corner of each panel.
